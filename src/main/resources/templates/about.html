About Me

Hello! my name is Mohammed Hattab, I'm passionate about the world of continuous learning and exploration. My journey as a developer has been defined by a deep love for acquiring new skills and knowledge.

This simple project, built using Spring Boot and Thymeleaf, represents more than just lines of code to me. It embodies the intricate journey a developer undertakes to bring an idea to life. From conception to production, every step mirrors the challenges and triumphs that shape the development process.

One of the highlights of this project was subjecting the code to a static scanner using CodeQL on GitHub. This meticulous analysis not only enhanced the code quality but also reflects my commitment to producing robust and secure solutions.

Deploying the project on AWS Elastic Beanstalk through AWS Pipelines marked another milestone. The seamless orchestration of this deployment pipeline showcased my proficiency in modern development practices and cloud technologies.

Security was a paramount concern during every phase of this project's lifecycle. Every decision, from architecture design to implementation, was made with a security-first approach.
